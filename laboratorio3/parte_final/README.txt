Este documento posee la descripcion de los tres ultimos programas
que se deben realizar en el laboratorio 3 del curso.

El primero "strcat2.cpp" esta relacionado con la funcion 'strcat', pero se realizo 
modificacion al codigo principal para que ahora funcione con punteros.

El segundo "strend2.cpp" esta relacionado con la comparacion que se puede
realizar entre dos strings, lo que hace es recibir dos strings digitados
por el usuario y dependiendo de la terminacion de cada uno, returna un 1 si son iguales, o un cero son distintas.

El tercero "strcopy.cpp" esta relacionado con la copia de el string apuntado por "source" dentro del arreglo
apuntado por "destination"
